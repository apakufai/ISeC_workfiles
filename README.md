# Individual Self-Exploration Card


Название:<br/> 
Бэк: ISC (Individual Self-exploration Card)<br/> 
Фронт: ИКС (Индивидуальная карта самоисследования)<br/>

#e29fa9 - светло-красный (цвет рамки отмеченного ответа)
